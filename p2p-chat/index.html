P2P Chat Application with PeerJS

Instructions

  1. Start the Application: Open the application in two separate browser windows or devices.
  2. Peer 1 (Host):
    • Your Peer ID is displayed below. Share this ID with Peer 2.
  3. Peer 2 (Client):
    • Enter Peer 1's ID into the "Connect to Peer ID" field and click "Connect".
  4. Chat: Once connected, both peers can send messages to each other.

Note: This setup uses PeerJS's default signaling server. For production use, consider hosting your own signaling server.
